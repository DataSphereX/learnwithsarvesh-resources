# Webinar Starter Kits

These are projectized, ready-to-run versions of webinar notebooks.

## Available starter kits

1. `resume-analyzer`
   - Batch resume parsing, skill-gap analysis, candidate ranking output.
2. `customer-support-chatbot`
   - FAQ RAG chatbot with persistent Chroma index and session memory.
3. `rag-pdf-summarizer`
   - PDF ingestion, FAISS indexing, and interactive document Q&A.

## How to use

1. Pick one project folder.
2. Follow that folder's `README.md`.
3. Configure `.env` using `.env.example`.
4. Run the provided scripts.

## Why this folder exists

The original notebooks are still useful for teaching. These starter kits are for students who want clean project structure and reusable code immediately.
