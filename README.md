# learnwithsarvesh-resources

The main objective is to share study materials and supporting resources that make it easier to learn, practice, and teach. This repo organizes quick references, interview prep, sample code, and webinar content in one place.

## What's inside
- `cheat-sheets/`: Condensed study notes and quick-reference guides.
- `interview-preparation/`: Questions, answers, and notebooks for interview practice.
- `sample-codes/`: Example scripts and snippets for demos or practice.
- `webinar-materials/`: Slides, handouts, and other webinar resources.

## How to use
- Browse folders for the topic you need.
- Copy snippets or notes for personal study.
- Submit improvements via PRs to keep resources current.
